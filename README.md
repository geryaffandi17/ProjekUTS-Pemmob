# Projek UTS Android Studio

# Deskripsi
Aplikasi ini dibuat menggunakan Android Studio dengan bahasa Kotlin.  
Project ini merupakan aplikasi pendaftaran seminar yang memiliki fitur login, registrasi, dan form pendaftaran.

# Fitur Aplikasi
- Login User
- Registrasi User
- Validasi Realtime
- Konfirmasi Password
- Pilih Jurusan
- Form Pendaftaran Seminar
- Menampilkan Hasil Data

# Halaman Aplikasi
1. LoginActivity
2. RegisterActivity
3. MainActivity
4. FormActivity
5. ResultActivity

# Teknologi yang Digunakan
- Android Studio
- Kotlin
- XML Layout
- SharedPreferences
